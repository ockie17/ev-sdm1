Konfigurasi Firebase

Masukkan konfigurasi Firebase project Anda. Data evaluasi akan tersimpan di Realtime Database dan dapat diakses oleh semua pengguna secara realtime.

1
Buka console.firebase.google.com → klik Add project → beri nama project → klik Continue hingga selesai.
2
Di sidebar kiri klik Build → Realtime Database → klik Create Database → pilih lokasi → pilih Start in test mode → Enable.
3
Klik ikon roda gigi (⚙️) → Project settings → scroll ke bawah → klik ikon </> → Register app → salin nilai dari firebaseConfig.
4
Tempelkan nilai konfigurasi ke kolom di bawah ini, lalu klik Simpan & Mulai.
Gagal terhubung ke Firebase. Periksa kembali konfigurasi Anda.

Konfigurasi disimpan di browser (localStorage) sehingga tidak perlu diisi ulang.
